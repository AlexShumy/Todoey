
# Todoey 
A fun todolist app to keep track of all your tasks

## Features

* Table views, UITableViewControllers
* Saving data in iOS fundamentals
* UserDefaults, Core Data and Realm
* Swipe gesture functionality using SwipeCellKit
* Gradient flow cells
